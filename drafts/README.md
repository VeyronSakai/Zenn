# Drafts Directory

このディレクトリは Zenn 記事作成エージェントの作業用ディレクトリです。

## 構造

```text
drafts/
  <slug>/
    outline.md   # 📋 Planner が作成するアウトライン
    draft.md     # ✍️ Writer が作成するドラフト
    review.md    # 🔍 Reviewer が作成するレビュー結果
```

## 注意

- このディレクトリの内容は作業用のため、Zenn には公開されない
- 最終版は `articles/` ディレクトリに出力される
- 必要に応じて過去の drafts を削除する
